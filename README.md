# Socket Test

### 최종 프로젝트 전 socket Test

 사용자 이름 입력 후 엔터 시 채팅 페이지로 이동 <br/>
 채팅 열기 클릭 시 작은 화면으로 채팅 방 리스트 보여줌 <br/>
 방 클릭 시 해당 방으로 이동하여 채팅 가능 <br/>
 x 클릭 시 채팅 방 리스트로 돌아감 <br/>
 채팅방 리스트에서 x 클릭 시 채팅창 닫힘 <br/>
